## Overview

- `grouped-view` – shows to-do list grouped with the specified key
- `eisenhower-view` – shows to-do list in an Eisenhower matrix

## Installation

Install with `pipx`, `pip` or `uv` accordingly:

```
pipx install vitodo
pip install vitodo
uv tool install vitodo
```
